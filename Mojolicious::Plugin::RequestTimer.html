Mojolicious::Plugin::RequestTimer

名前

Mojolicious::Plugin::RequestTimer - リクエストタイマープラグイン

使い方

# Mojolicious
$self->plugin('RequestTimer');

# Mojolicious::Lite
plugin 'RequestTimer';

説明

Mojolicious::Plugin::RequestTimerは、リクエスト時の情報を収集して記録するためのプラグインです。 これはコアプラグインなので、いつでも使用可能であり、新しいプラグインを作成する方法を学ぶための良いコード例になります。

メソッド

Mojolicious::Plugin::RequestTimerは、Mojolicious::Pluginのすべてのメソッドを継承し、以下の新しいメソッドを実装しています。

register

$plugin->register;

Mojoliciousアプリケーションにプラグインのフックを登録します。
