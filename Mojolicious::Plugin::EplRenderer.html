名前

Mojolicious::Plugin::EPLRenderer - EPLレンダラプラグイン

使い方

# Mojolicious
$app->plugin('EPRenderer');
$app->plugin(EPRenderer => {name => 'foo'});
$app->plugin(EPRenderer => {name => 'bar', template => {line_start => '.'}});

# Mojolicious::Lite
plugin 'EPRenderer';
plugin EPRenderer => {name => 'foo'};
plugin EPRenderer => {name => 'bar', template => {line_start => '.'}};

説明

Mojolicious::Plugin::EPLRendererはeplテンプレートのレンダラです。 eplテンプレートはほとんど生のMojo::Templateです。 これはコアのプラグインで、いつでも有効になっており、 そのコードは新しいプラグインを構築する ことを学ぶための良いサンプルです。

デフォルトで利用可能なプラグインの一覧はMojolicious::Pluginsのプラグインの項目を見てください。

メソッド

Mojolicious::Plugin::EPLRendererは Mojolicious::Pluginからすべてのメソッドを継承しており、 次の新しいメソッドを実装しています。

register

$plugin->register;

Mojoliciousアプリケーションにレンダラを登録します。

参考

Mojolicious, Mojolicious::Guides, http://mojolicio.us.
