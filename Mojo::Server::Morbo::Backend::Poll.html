Mojo::Server::Morbo::Backend::Poll - Morboのデフォルトのバックエンド

使い方

use Mojo::Server::Morbo::Backend::Poll;

my $backend = Mojo::Server::Morbo::Backend::Poll->new;
if (my $files = $backend->modified_files) {
  ...
}

説明

Mojo::Server::Morbo::Backend::PollはMojo::Server::Morboのデフォルトのバックエンドです。

属性

Mojo::Server::Morbo::Backend::Pollは、Mojo::Server::Morbo::Backendからすべての属性を継承しています。

メソッド

Mojo::Server::Morbo::Backend::PollはMojo::Server::Morbo::Backendからすべてのメソッドを継承しており、次の新しいメソッドを実装しています。

modified_files

my $files = $backend->modified_files;

Check file size and mtime to determine which files have changed, this is not particularly efficient, but very portable.

ファイルサイズとmtimeをチェックしてどのファイルが変更されたかを判断します。 効率的ではありませんが、非常に移植性があります。

# 修正されたすべてのファイル
say for @{$backend->modified_files};
