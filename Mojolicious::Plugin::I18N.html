Mojolicious::Plugin::I18N - 国際化プラグイン

使い方

# Mojolicious
$self->plugin('I18N');
% languages 'de';
%=l 'hello'

# Mojolicious::Lite
plugin I18N => {namespace => 'MyApp::I18N'};
%=l 'hello'

# Lexicon
package MyApp::I18N::de;
use Mojo::Base 'MyApp::I18N';

our %Lexicon = (hello => 'hallo');

1;

説明

Mojolicous::Plugin::I18n はLocale::MaketextのサポートをMojoliciousに追加します。

このプラグインを有効にした上で行うことは、必要な分の辞書クラスを追加することだけです。 通常、言語はAccept-Languagesリクエストヘッダから自動的に検出されます。

デフォルトで次のようなコード自動的に生成されるので、 このプラグインでたくさんのタイピングを省略することができます。

# $self->plugin('I18N');
package MyApp::I18N;
use base 'Locale::Maketext';
package MyApp::I18N::en;
use base 'MyApp::I18N';
our %Lexicon = (_AUTO => 1);
1;

生成されるコードの名前空間とデフォルトの言語はそれぞれのオプションに影響されます。 デフォルトの語集クラスは、まだそれが存在しない場合にだけ生成されます。

オプション

default

# Mojolicious::Lite
plugin I18N => {default => 'en'};

デフォルトの言語。

namespace

# Mojolicious::Lite
plugin I18N => {namespace => 'MyApp::I18N'};

語集の名前空間。デフォルトはアプリケーションクラスに続く::I18Nになります。

ヘルパー

l

%=l 'hello'
$self->l('hello');

文の翻訳。

languages

% languages 'de';
$self->languages('de');

言語を切り替えます。

メソッド

Mojolicious::Plugin::I18Nは、Mojolicious::Pluginのすべてのメソッドを継承し、以下の新しいメソッドを実装しています。

register

$plugin->register;

プラグインのフックとヘルパーをMojoliciousアプリケーションに登録します。

参考

Mojolicious, Mojolicious::Guides, http://mojolicio.us.
