Mojolicious::Plugin::Charset - 文字セットプラグイン

使い方

# Mojolicious
$self->plugin(Charset => {charset => 'Shift_JIS'});

# Mojolicious::Lite
plugin Charset => {charset => 'Shift_JIS'};

説明

Mojolicious::Plugin::Charsetはデフォルトの 文字セットを簡単に設定する Mojoliciousのすべての層のためのエンコーディングを 行うためのプラグインです。

デフォルトで利用可能なプラグインの一覧はMojolicious::Pluginsのプラグインの項目を見てください。

オプション

charset

# Mojolicious::Lite
plugin Charset => {charset => 'Shift_JIS'};

アプリケーションの文字セット。

メソッド

Mojolicious::Plugin::Charsetは Mojolicious::Pluginからすべてのメソッドを継承しており、 次の新しいメソッドを実装しています。

register

$plugin->register;

Mojoliciousアプリケーションの中のbefore_dispatchフックで登録され、いくつかのデフォルト値を変更します。

参考

Mojolicious, Mojolicious::Guides, http://mojolicio.us.
