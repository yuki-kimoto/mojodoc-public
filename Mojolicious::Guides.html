チュートリアル

Mojolicious でウェブアプリケーションの開発を始める本当に早く楽しい方法は Mojolicious::Lite のチュートリアルにあります。 このチュートリアルで学ぶほとんどすべてのことは、普通のMojoliciousアプリケーション で利用することができ、ガイドを読む前に学習するためのものになっています。 絶対に見るべきです。

ガイド

  • Mojolicious::Guides::Routing

    簡単で楽しい Mojolicious ルータへの入門。

  • Mojolicious::Guides::Rendering

    Mojolicious レンダラでの内容生成。

  • Mojolicious::Guides::Growing

    ゼロからの Mojolicious::Lite の原型の開始、そしてそれを 良く構造化された Mojolicious アプリケーションへ育てること。

  • Mojolicious::Guides::Cookbook

    Mojolicious での料理、あらゆる風味のためのレシピ。

  • Mojolicious::Guides::CodingGuidelines

    進行中のMojolicious開発に参加しよう。

  • Mojolicious::Guides::FAQ

    よくある質問と正しい回答。

ハイライト

Mojolicious と Mojolicious::Lite は多くの部品、 つまり独立に使用可能である小さな構成ブロックが集約されたものです。 これらは最も顕著なものです。

  • Mojo::UserAgent

    完全な機能を備えた非同期 I/O HTTP 1.1 および WebSocket ユーザエージェント。

  • Mojo::DOM

    とても楽しい、CSS3 セレクタをサポートした最小限の XML/HTML5 DOM パーサ。

  • Mojo::JSON

    正しく動作する最小限の JSON 実装。

  • Mojo::Server::Daemon

    高度にポータブルな非同期 I/O HTTP 1.1 および WebSocket サーバ。 開発とテストには申し分ありません。

  • Mojo::Server::Prefork

    完全な機能を備え、UNIXに最適化された、 プレフォーキング非同期 I/O HTTP 1.1 および WebSocket サーバ。 Mojo::Server::Hypnotoadを通して、 停止時間なしでのソフトウェア更新(ホットデプロイメント)をサポートします。

  • Mojo::Server::CGI, Mojo::Server::PSGI

    そのままで CGI, FastCGI, PSGI を透過的にサポート。

  • Mojo::IOLoop

    複数のリアクターバックエンドをサポートした最小限のイベントループ。

  • Mojo::Template

    とても Perl 的、かつ最小限のテンプレートシステム。

  • Test::Mojo

    ウェブアプリケーションのためのテスト駆動開発ツールキット。

  • ojo

    上記全ての利用をワンライナーでお楽しみ下さい。

より学ぶには

Mojolicious::Guidesを学び続けてください。またMojolicious wikiを見ることもできます。wikiには多くのユーザーによる多くのドキュメントやサンプルがあります。
